I design complex B2B SaaS products — dense data, multi-step workflows, users who have zero patience for confusion. My work has been adopted by **Groww** and **ShareChat**, and helped boost client sales by **30%+**.

But I don't just design. I build.

I shipped **LoanLens** — a financial simulation tool for Indian users — using Claude and Next.js. Every design decision, every line of code, every deployment. That experience changed how I think about design forever.
